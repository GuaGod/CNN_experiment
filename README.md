### 实验索引
实验名称： 不同架构CNN网络的运行表现 </br>
开始时间： 2022年4月13日    </br>
实验内容简介：实验数据基于CIFAR_10数据集，测试不同架构的CNN网络在不
同的超参数下进行图片分类任务的性能指标。主要测试VGG架构，RES架构以及
Inception架构。实验的分析分为横向对比不同架构和纵向对比不同参数两个维度。 </br>
